Step 03

국민 혜택

몰랐던 혜택을, 지금 바로 신청하게

Before

기존 방식
  • 여러 사이트 직접 검색
  • 자격 요건 이해 어려움
  • 신청 링크·전화번호 직접 찾아야
  • 디지털 취약계층은 포기

After

드리미 사용 시
  • 사진 한 장 → 평균 20초
  • 쉬운말로 자동 번역·안내
  • 전화·길찾기 버튼으로 즉시 연결
  • 큰 글씨 + TTS — 어르신도 혼자 가능

발전방안

즉시 보완 1~2주
  • Corpus 44건 → 무제한 확장 (아동·임산부·의료 추가)
  • OCR 정확도 개선 (Gemini Vision API 연동)
  • 위저드 입력 전체 구현 (상황 태그 선택지)
  • 실서버 배포 (로컬 → 클라우드)
현업 도입
  • 복지로 API 연동 → 실시간 수급 자격 확인
  • 관리자 대시보드 운영 → 사용량 통계 관리
  • 다국어 지원 (한국어·베트남어·중국어)
  • 고용24 연계 → 일자리+복지 통합 안내
